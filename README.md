# SMS
Secure Microfinance System

## Overview
SMS (Secure Microfinance System) is a modern, privacy-focused, and modular web platform for managing microfinance operations. Built with Next.js, TypeScript, and Tailwind CSS, it emphasizes data security, auditability, and extensibility for financial inclusion organizations.

## Key Features
- Customer and loan management
- User roles and permissions
- Audit logging and activity tracking
- Data privacy: encryption and masking utilities
- Notifications and admin dashboard
- Modular, service-oriented architecture
- Beautiful, responsive UI (Tailwind CSS)
- Extensible middleware and service layers


## Purpose
To provide a secure, extensible, and easy-to-deploy microfinance management system for organizations prioritizing data privacy and modern web standards.

## Scope
- Customer onboarding and management
- Loan origination and tracking
- User authentication and authorization
- Audit logs and activity monitoring
- Privacy and data protection utilities
- Admin and notification modules

## Limitations
- No native mobile app (web only)
- Not multi-tenant by default
- Lacks advanced analytics and legacy system integration
- Not blockchain-enabled

## Getting Started
1. Clone the repository
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm dev`
4. Access the app at `http://localhost:3000`

## Folder Structure
- `app/` - Main application pages and API routes
- `components/` - UI components
- `lib/` - Core business logic and utilities
- `database/` - Database schema
- `hooks/` - Custom React hooks
- `public/` - Static assets
- `styles/` - Global styles
